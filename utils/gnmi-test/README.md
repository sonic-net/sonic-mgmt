gnmitest framework helps with testing the gNMI target based on OC models 
against specification and also helps collect performance numbers.

gnmi test provides ways to test the get/set/subscribe functionality. 
1. Validates them against OC schema.
2. Its an extension to the existing opensource framework gnmitest. https://github.com/openconfig/gnmitest.git
3. Now performance numbers are added and exported as part of the report. 

Please Refer https://wiki.cisco.com/display/GPLATFORM/gNMI+Performance+Tool for more details.
